= ABL eM4 charging station

Implementation of the ABL eM4 electric vehicle charging station.

== Compatibility

** eM4 Controller (Single/Twin)
** eM4 Extender (Single/Twin)

(Note: The Extender uses a slightly different protocol and Firmware version)

Implemented Natures:
