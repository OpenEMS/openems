= BGE-TECH DRT428M-2 Meter

Three-phase bidirectional electricity meter for consumption and production measurement via Modbus protocol.

== Overview

This bundle provides integration with the B+G E-Tech DRT428M-2 meter, a bidirectional three-phase electricity meter with calibrated accuracy for grid-connected installations. The component communicates with the meter through the Modbus TCP protocol, enabling real-time energy monitoring in both consumption and production directions.

The DRT428M-2 is designed for DIN rail mounting and supports S0 impulse output and RS485 communication. It measures both imported and exported energy, making it suitable for installations with solar inverters or battery systems.
