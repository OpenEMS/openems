= KACO blueplanet gridsave

Three-phase battery inverter with integrated grid management and safety features.

== Overview

This bundle provides integration with the KACO blueplanet gridsave battery inverter family, a line of three-phase AC/DC converters designed for residential and commercial energy storage applications. The inverters support SunSpec Modbus protocol for advanced communication and grid synchronization with configurable grid code settings.

KACO blueplanet gridsave is suitable for solar battery storage, peak shaving, and residential energy management systems requiring precise grid synchronization and safety compliance.

== Supported Devices

.KACO blueplanet gridsave Series
* **KACO blueplanet gridsave 50.0 TL3**: 50 kW three-phase inverter
* **KACO blueplanet gridsave 92.0 TL3**: 92 kW three-phase inverter

.Common Features
* **Type**: Three-phase battery inverter
* **Communication**: Modbus TCP/RTU (SunSpec)
* **Typical Use**: Battery energy storage, solar integration
* **Grid Connection**: Three-phase AC grid
* **Grid Codes**: Configurable (VDE-AR-N 4105, etc.)
* **Documentation**: https://www.kaco.de/[KACO icon:external-link[]]

== Components

This bundle implements the following OpenEMS Components:

=== <<_kaco_gridsave,KACO blueplanet gridsave>>

*Name*: Battery-Inverter KACO blueplanet gridsave

*Factory-PID*: `Battery-Inverter.Kaco.BlueplanetGridsave`

.Implemented Natures/Interfaces
* BatteryInverterKacoBlueplanetGridsave
* ManagedSymmetricBatteryInverter
* SymmetricBatteryInverter
* ModbusComponent
* ModbusSlave
* OpenemsComponent
* TimedataProvider
* StartStoppable
* BatteryInverterErrorAcknowledge

*Description*: SunSpec Modbus interface for KACO blueplanet gridsave battery inverters. Manages bidirectional AC/DC power flow, monitors inverter status, temperature, and grid conditions. Supports automatic start/stop control, grid code compliance, and watchdog timer for safety.

.*Configuration*:
* `id` (String): Component ID for unique identification (default: "batteryInverter0")
* `alias` (String): Human-readable alias for the component
* `enabled` (Boolean): Enable/disable this component (default: true)
* `startStop` (StartStopConfig): Start/stop behavior - AUTO, ALWAYS_ON, or ALWAYS_OFF (default: AUTO)
* `activateWatchdog` (Boolean): Enable watchdog timer for safety (default: true)
* `gridCode` (GridCode): Grid code setting for compliance (default: VDE_4105)
* `modbus_id` (String): ID of the Modbus bridge to use (default: "modbus0")
