= Soltaro Battery Rack

Modular lithium battery storage system with multiple hardware versions.

== Overview

This bundle provides integration with the Soltaro battery rack system, a scalable lithium-based energy storage solution offering both single-rack and clustered configurations. Multiple hardware versions (A, B, C) are supported to accommodate different system architectures and performance requirements.

Soltaro batteries are suitable for commercial energy storage, peak shaving, demand response, and renewable energy time-shifting applications.

== Supported Devices

.Soltaro Single Rack Systems
* **Version A**: Legacy single-rack configuration
* **Version B**: Mid-generation single-rack system
* **Version C**: Latest generation single-rack system
* **Typical Use**: Residential/small commercial
* **Scalability**: Single unit or stacked configurations

.Soltaro Cluster Systems
* **Version B**: Multi-module clustered configuration
* **Version C**: Latest generation clustered system
* **Typical Use**: Commercial/industrial energy storage
* **Scalability**: Multiple racks coordinated as single system

.Common Features
* **Type**: Modular lithium battery storage
* **Communication**: Modbus TCP/RTU
* **Typical Use**: Energy storage, peak shaving, arbitrage
* **Documentation**: https://www.soltaro.com/[Soltaro icon:external-link[]]

== Components

This bundle implements the following OpenEMS Components:

=== <<_soltaro_single_a,Soltaro Single Rack Version A>>

*Name*: BMS Soltaro Single Rack Version A

*Factory-PID*: `Bms.Soltaro.SingleRack.VersionA`

.Implemented Natures/Interfaces
* Battery
* ModbusComponent
* OpenemsComponent
* EventHandler
* ModbusSlave
* StartStoppable

*Description*: Modbus interface for Soltaro single-rack battery system Version A. Manages charging and discharging, monitors state of charge, temperature, and cell health.

*Configuration*:
.Configuration Parameters
* `id` (String): Component ID (default: "battery0")
* `alias` (String): Human-readable alias
* `enabled` (Boolean): Enable/disable (default: true)
* `startStop` (StartStopConfig): Start/stop behavior (default: AUTO)
* `modbus_id` (String): Modbus bridge ID (default: "modbus0")
* `modbusUnitId` (Integer): Modbus Unit-ID (default: 0)
